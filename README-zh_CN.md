## ⌨️ 本地开发

克隆到本地开发:

```bash
$ git clone https://github.com/magicsli/postson.git
$ cd postson
$ yarn install
$ yarn start


/* kangyan大佬vue3.0 版原址: */
$ https://github.com/kangyana/postson.git
```

打开浏览器访问 http://127.0.0.1:3000
